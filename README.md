# email.FTP
tugas UTS Program Jaringan

Aplikasi sisi server diinisialisasi dengan menjalankan file web_server.py melalui terminal. 
Setelah berjalan, terminal akan menampilkan alamat IP lokal (Host Address), contohnya http://192.168.18.132:5000. Klien dapat mengakses alamat tersebut secara langsung melalui web browser, dengan prasyarat bahwa perangkat Klien dan Server terhubung dalam satu jaringan Wi-Fi (Local Area Network) yang sama.
